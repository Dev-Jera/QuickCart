# QuickCart
Fullstack e-commerce platform built with Django REST Framework, React, Redux, PostgreSQL, and Docker. Features include product catalog, shopping cart, orders, role-based access (admin/customer), CI/CD, testing, and a responsive UI with modern UX principles.
